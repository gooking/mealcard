# 会员卡（饭卡）储值消费系统

方便用户自助充值消费、购买会员享受消费折扣、方便查看消费记录、查看动态和最新通知公告，助力私域流量营销

## 详细配置/使用教程

本项目是 uniapp 项目，使用 `HBuilder X` 开发工具打开，即可运行查看效果

1. 免费注册后台账号 [https://admin.s2m.cc/](https://admin.s2m.cc/)
2. 克隆测试数据，登录后台-工厂设置-数据克隆-将别人的数据克隆给我，输入商户ID `164` ，点击完成克隆后F5刷新网页
3. 根目录的 `config.js` 文件，请将其中的 `subDomain` 和 `merchantId` 配置项，修改为自己的专属域名和商户ID （登录后台-工厂设置-商户信息 处可直接查看）
4. 打开 `HBuilder X` 开发工具，点击 `文件` -> `打开项目`，选择本项目目录，即可打开项目
5. 点击 `运行` -> `运行到浏览器`，即可在浏览器中查看效果
